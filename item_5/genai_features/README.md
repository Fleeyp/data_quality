## Primeiramente, na seguinte url:
https://openai.com/pt-BR/index/openai-api/

. Crie, ou recupere sua chave da API OpenAI;
. Substitua, no arquivo ../product_feature_extraction, "gen_ai_api_key_here", pela chave criada/recuperada.

## Para rodar, lançe no terminal, na raiz do projeto:
python3 item_5/genai_features/product_feature_extraction

## Importante:
Comando que deve ser rodado para instalar os requirements necessários:
pip install -r requirements.txt
